### Description

This mini application allows users to create, display, and delete text notes directly in the browser.

### Functionality

* Users can write a note in an input field.
* Notes are dynamically rendered in the page using DOM manipulation.
* Notes can be deleted individually.
* Notes are stored in `localStorage` to persist after page reloads.

### Purpose

Practice DOM manipulation, event handling, and browser storage using vanilla JavaScript.
